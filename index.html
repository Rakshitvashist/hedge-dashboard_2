Institutional Logic & Hedge Defense Layers

1. Base SIM (Portfolio Optimization)

What we do: This is the foundational selection. We calculate individual stock Betas and Residual Variance (σ²) using the Single Index Model (SIM).

Logic: Stocks are ranked by their Excess Return to Beta (ERB). We calculate a mathematical cut-off (Ci) and select only those stocks that provide "alpha" above this threshold.

Constraint: We cap individual stock weights at 10% to prevent over-concentration in any single name.

2. ST Filter (Momentum Defense)

What we do: We apply a momentum filter to the Base portfolio.

Logic: We check the Supertrend (10, 3) indicator on a monthly basis for every stock.

Result: We only keep stocks that are in a Bullish state (Trend = 1). If a stock's momentum turns bearish, it is immediately removed from this layer to avoid "catching a falling knife."

3. EMA Filter (Trend Defense)

What we do: We apply a long-term trend filter to the Base portfolio.

Logic: We calculate the 40-period Exponential Moving Average (EMA).

Result: We only retain stocks trading above their EMA-40. This ensures the portfolio is always aligned with the primary long-term trend and ignores stocks in structural downtrends.

4. COMBO Filter (High-Conviction Layer)

What we do: We implement a "Double-Lock" validation system.

Logic: A stock is only selected if it passes BOTH the Supertrend (ST) and the EMA-40 filter simultaneously.

Result: This layer significantly reduces volatility and drawdowns by ensuring we only hold stocks that have both strong momentum and a confirmed long-term trend.

5. ULTRA Layer (Alpha Concentration)

What we do: This is a concentrated version of the COMBO selection.

Logic: It focuses on the elite performers that have cleared all previous institutional filters.

Result: It serves as the baseline for the most aggressive hedging strategies when market breadth starts to deteriorate.

6. COMBO + Hedge (Active Beta Hedging)

What we do: We actively neutralize market risk using NIFTY Futures.

Logic: We monitor Nifty's Multi-Timeframe (MTF) alignment across 15m, 30m, 1h, and 1d timeframes.

Result: When all four timeframes turn Bearish, we "Hedge" the portfolio by shorting Nifty Futures proportional to the portfolio's Beta. This protects the gains of the COMBO stocks during market crashes.

7. ULTRA Defense (Aggressive Multiplier Hedge)

What we do: We implement an aggressive "Tail-Risk" protection mechanism.

Logic: We monitor Market Breadth (the ratio of stocks passing the COMBO filter vs. the Base selection).

Result: If breadth falls below 50% (indicating a systemic breakdown), we apply a 2.0x Beta multiplier to the Nifty short position. This provides "Super-Hedging" to protect the capital during severe liquidations or "Black Swan" events.
